# README

### 代码使用环境

Windows/Linux + Shell

### 使用步骤：

1. 从https://zenodo.org/records/10209837 下载CC数据集，将Healthy和COPD两个数据集放在`data/raw_data`下

2. 安装所用的python环境：

   ```shell
   pip install -r requirement.txt
   ```

3. 运行run.sh：

   ```shell
   cd ..
   sh run.sh
   ```

### 相关说明

1. 运行结果：存储在`./log`和`./saved`文件夹中
